# TomROverview
Basic stats for TomR

#Setting up the System:
1. Clone https://github.com/rchakra3/TomROverview:
   *git clone "https://github.com/rchakra3/TomROverview"
   *cd TomROverview
2. On the monitoring node:
   npm install --save
   node app.js
2. On all the datanodes:
   cd client
   npm install --save
   node app.js
